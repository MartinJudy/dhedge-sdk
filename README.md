# dhedge-sdk


## Installation

```bash
npm install dhedge-sdk
```

## Setup .env

Copy the `.env.example` file to `.env` in the root of your project and fill out the missing information:

```bash
MNEMONIC=""
ACCOUNT_ID=""
PROVIDER=""
FACTORY_ADDRESS=""
GAS_ESTIMATE_PADDING="20"
```
